# Unity Blueprint

This folder contains the Unity-oriented gameplay architecture for `NH6 Pursuit`.

It is not a generated Unity project. It is a clean blueprint intended for migration into a real Unity setup.

## Included

- `Assets/Scripts/Core/RaceSessionState.cs`
- `Assets/Scripts/Core/GameBootstrap.cs`
- `Assets/Scripts/Systems/PlayerCarController.cs`
- `Assets/Scripts/Systems/TrafficCoordinator.cs`
- `Assets/Scripts/Systems/PursuitDirector.cs`
- `Assets/Scripts/Systems/RoadStreamSystem.cs`
- `Assets/Scripts/Data/RoadThemeProfile.cs`

## Intended engine target

- Unity 6+
- URP or HDRP depending on performance and art goals
